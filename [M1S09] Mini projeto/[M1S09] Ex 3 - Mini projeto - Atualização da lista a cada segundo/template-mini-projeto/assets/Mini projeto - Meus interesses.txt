Mini projeto - Atualização da lista a cada segundo.

-Nesta tarefa, vamos garantir que nossos dados estejam sempre atualizados. 
Para isso, utilize o setInterval para recuperar a lista e atualizá-la a 
cada 1 segundo. Basta usar a função do primeiro exercício.

-Certifique-se de que a função criada está "limpando" os dados. 
-Caso contrário, antes de adicionar os interesses, inclua um código 
para limpar as informações. Você pode fazer isso pegando o elemento 
da lista e definindo o innerHTML como vazio, ou seja, 
elemento.innerHTML = "" .

‌Observação: 

-Lembre-se de que o segundo parâmetro do setInterval, 
o atraso, é definido em milissegundos.
